# Rick and Morty API – Buscador de Personajes

Trabajo Práctico N.º 2 – Consumo de API REST | Desarrollo y arquitecturas web

Rodolfo Meroi

Página estática que consume la API REST de [rickandmortyapi.com](https://rickandmortyapi.com/) para buscar y listar personajes, con filtros por nombre, status, especie, tipo y género, y paginación de resultados.
